# QuickShulker

View and interact with shulker box contents directly from your inventory.

## Features

- **Hover preview** — hover a shulker to see its contents in a compact 3×9 grid
- **Extract** — right-click to take the selected item out
- **Insert** — left-click with an item in hand to place your entire held stack
- **Scroll selection** — scroll to cycle through shulker slots
- **Slot indicators** — coloured overlays show if you can insert (yellow), stack (green), or if it's full (red)
- **Server-validated** — all actions checked server-side

## Requirements

- Minecraft 26.1.2
- Fabric Loader ≥0.19.2
- Fabric API
